## ✨ Key Features
- Authentication — Sign up, Sign in, Forgot password (OTP)
- Smart Task Creation — Title, deadline, priority, tags, AI suggestions
- Kanban Task Board — Drag and drop, multi-view (List/Grid/Calendar)
- AI Recommendations — Auto-sort, focus-time, smart notifications
- Daily Planner — Time block view, reschedule via drag and drop
- Team Collaboration — Invite members, assign tasks, in-app chat, shared notes
- Blockchain Task Verification — Immutable logs, wallet connect, signed approvals
- Google Calendar Sync & iCal export
- Subscription Tiers — Silver, Gold, Diamond with payment gateway
- Data Analytics Dashboard — Weekly charts, focus analysis, productivity reports
- AI Voice Assistant — Add tasks, reschedule and set reminders by voice
- Offline Mode — Local sync, auto-sync on reconnection
- Admin Panel — Manage users, subscriptions, blockchain ledger viewer
- Security — AES encryption, GDPR consent, session timeout
